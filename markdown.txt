# 三、Host + Server Browser + Join Widgets

将任何资产在 UE5 中移动的时候，有时我们会获得 redirector，redirector 在使用的过程中可能会变得混乱，因为有可能有错误的引用，因此为了确保始终有正确的引用 reference，我们可以右键 content 选择 Update Redirector References
